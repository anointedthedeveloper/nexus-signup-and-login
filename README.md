NexusHub - Login & Signup Pages (v1)
A modern, animated login and signup page with smooth transitions, hover effects, and ripple buttons.

Features
Responsive Design: Works seamlessly on all devices

Smooth Animations: Elegant transitions and hover effects

Ripple Button Effects: Interactive feedback on button clicks

Form Validation: Real-time input validation with visual feedback

Password Strength Indicator: Visual password strength meter

Google Sign-in Option: Ready for OAuth integration

Floating Shapes Animation: Dynamic background elements

Page Transitions: Smooth navigation between pages

Getting Started
Clone or download the project files

Open login.html in your browser to see the login page

Click "Sign up" to navigate to the signup page

Test the form validation and animations

Customization
Colors
Modify the CSS variables in style.css:

css
/* Primary gradient */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
Background Image
Replace the Unsplash image URL in style.css:

css
background: url('your-image-url-here') center/cover no-repeat;
Animations
Adjust animation timing and easing in both CSS files.
